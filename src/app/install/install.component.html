Install Kaiac

To install Kaiac you just need an Ubuntu virtual machine and use the following commands.

Make sure you have some utilities:

$ sudo apt-get update && sudo apt-get install -y gnupg software-properties-common

Choose kaiac version you want:

$ VERSION="0.0.7"

Add the Kaiac GPG key:

$ wget -O- http://repo-kaiac-prod.skyscaledev.com/v$VERSION/gpg | gpg --dearmor | sudo tee /usr/share/keyrings/kaiac-keyring.gpg

Add the official Kaiac Linux repository:

$ echo "deb [arch=amd64 signed-by=/usr/share/keyrings/kaiac-keyring.gpg] http://repo-kaiac-prod.skyscaledev.com/v$VERSION/apt-repo stable main" 
 | sudo tee /etc/apt/sources.list.d/kaiac.list

Download and refresh package index:

$ sudo apt update

Finally install kaiac:

$ sudo apt-get install kaiac

Verify Kaiac

To verify your installation, use the following command:

$ kaiac env

The result should look like:

        VERSION   = 0.0.7
        STATE     = bucket[Not defined] region[us-west-2] profile[default]
        RESOURCES = region[us-west-2] profile[default]
        KVBOOK    = Not defined
        MODULE    = Not defined
        CONTEXT   = namespace[Not defined] name[Not defined] stage[Not defined]
