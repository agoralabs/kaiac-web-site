Install KaiaC

1) Make sure you have a Debian virtual with AWS CLI :

KaiaC is released as a Debian package, so to work properly, KaiaC needs the following.

  • A Debian or Ubuntu virtual machine : You can use Vagrant and VirtualBox to easily create a virtual machine inside your current PC. Click on this link How-use-Vagrant-and-VirtualBox. For Ubuntu, you can use this Vagrant Box
  • AWS CLI installed on the virtual machine : To install or update to the latest version of the AWS CLI, Click on this link Getting-started-install

2) Make sure you have some utilities : 

$ sudo apt-get update && sudo apt-get install -y gnupg software-properties-common

3) Choose KaiaC version you want :

$ VERSION="0.0.9"

4) Add the KaiaC GPG key :

$ wget -O- http://repo-kaiac-prod.skyscaledev.com/v$VERSION/gpg | gpg --dearmor | sudo tee /usr/share/keyrings/kaiac-keyring.gpg

5) Add the official KaiaC Linux repository :

$ echo "deb [arch=amd64 signed-by=/usr/share/keyrings/kaiac-keyring.gpg] http://repo-kaiac-prod.skyscaledev.com/v$VERSION/apt-repo stable main" | sudo tee /etc/apt/sources.list.d/kaiac.list

6) Download and refresh package index :

$ sudo apt update

7) Finally install KaiaC :

$ sudo apt-get install kaiac

Verify KaiaC

To verify your installation, use the following command :

$ kaiac env

The result should look like :

        VERSION     = 0.0.9
        STATE       = bucket[Not defined] region[us-west-2] profile[default]
        RESOURCES   = region[us-west-2] profile[default]
        COMPOSITION = Not defined
        KVBOOK      = Not defined
        MODULE      = Not defined
        CONTEXT     = namespace[Not defined] name[Not defined] stage[Not defined]
