Infrastructure As Code made simpler

Kaiac is a tool made to simplify Infrastructure As Code. Kaiac is built on top of Terraform and allow you to create cloud resources just with few key values.

The key features of Kaiac are:

  • Infrastructure as Key Value: Infrastructure is described using a simple configuration syntax. This allows a blueprint of your datacenter to be versioned and treated as you would any other code. Additionally, infrastructure can be shared and re-used.

  • Terraform features: Kaiac mostly used Terraform in backgroup so you can inherit all the powerfull Terraform features like execution plans. For more information on Terraform, refer to the What is Terraform? page on the Terraform website.
