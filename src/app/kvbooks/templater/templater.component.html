KvBook Templater

The Templater KvBook allow you to parse a file containing placeholders with KaiaC.

Design

This is what you can create with a Templater KvBook

Key/Value parameters

# Name Example value Description
1 GL_AWS_REGION us-west-2 The AWS region your resources will be deployed
3 GL_AWS_CRED_FILE_PATH ~/.aws/credentials The AWS credentials file path
4 GL_AWS_CRED_PROFILE default The AWS credentials profile
5 GL_KAIAC_MODULE templater The name of kaiac module to run
6 GL_NAMESPACE tpl001 The application global resources namespace
7 GL_NAME demo1 The application name
8 GL_STAGE staging The application stage name
9 GL_INPUT_CONFIG_FILE /templater/001_runbook_cloud_oregon_templater_alfresco_config.json The environment variables config json file. For example: 
{{ '{' }}                                
    "TF_VAR_ENV_APP_GL_AWS_REGION": "us-west-2",
    "TF_VAR_ENV_APP_GL_NAMESPACE": "alfresco",
    "TF_VAR_ENV_APP_GL_STAGE": "prod",
    "TF_VAR_ENV_APP_GL_ROOT_DOMAIN": "skyscaledev.com",
    "TF_VAR_ENV_APP_BE_LOCAL_PORT": "8080",
    "TF_VAR_ENV_APP_BE_EXPOSED_PROTOCOL": "http",
    "TF_VAR_ENV_APP_BE_UPDATE_DNS_IF_RESTART": "YES",
    "TF_VAR_ENV_APP_HOSTED_ZONE_ID": "Z0017173R6DN4LL9QIY3",
    "TF_VAR_ENV_APP_WAIT_INTERVAL": "5",
    "TF_VAR_ENV_APP_CHANNEL_DEFAULT": "API",
    "TF_VAR_ENV_APP_API_SEND_COMMAND_URL": "https://lamb001.skyscaledev.com/send_command",
    "TF_VAR_ENV_APP_TAG_NAME": "Deployment",
    "TF_VAR_ENV_APP_LOCAL_FOLDER_PREFIX": "/kaiac",
    "TF_VAR_ENV_APP_MONITORING_CONTEXT": "alfresco/service/api/server",
    "TF_VAR_ENV_APP_MONITOR_RETRY_COUNTER_DEFAULT": "3",
    "TF_VAR_ENV_APP_LOG_FILE_OUTPUT": "/root/manager.log"
{{ '}' }}
10 GL_INPUT_TEMPLATE_FILE /templater/manager.sh.template The input template file
11 GL_OUTPUT_GENERATED_FILE /root/manager.sh The output file

Try

You can use this KvBook to try Templater with KaiaC :
GL_AWS_REGION="us-west-2"
GL_AWS_CRED_FILE_PATH="~/.aws/credentials"
GL_AWS_CRED_PROFILE="default"
GL_KAIAC_MODULE="templater"
GL_NAMESPACE="tpl001"
GL_NAME="demo1"
GL_STAGE="staging"
GL_INPUT_CONFIG_FILE="/templater/001_runbook_cloud_oregon_templater_alfresco_config.json"
GL_INPUT_TEMPLATE_FILE="/templater/manager.sh.template"
GL_OUTPUT_GENERATED_FILE="/root/manager.sh"
