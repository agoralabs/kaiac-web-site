KvBook VMOnly

The VMOnly KvBook allow you to create an EC2 instance with KaiaC. This EC2 instance will be created in your default region, default vpc and default public subnet

Design

This is what you can create with a VMOnly KvBook

Key/Value parameters

# Name Example value Description
1 GL_KAIAC_MODULE vmonly The name of kaiac module to run
2 GL_NAMESPACE vm001 The application global resources namespace
3 GL_NAME demo1 The application name
4 GL_STAGE staging The application stage name
5 BE_PATH_EC2_PUBKEY ~/.ssh/id_rsa.pub The ec2 instance public key path
6 BE_EC2_TYPE t2.micro The EC2 instance type
7 BE_EC2_VOLUME_SIZE 20 The EC2 instance root volume size en giga
8 BE_AMI_NAME ubuntu/images/hvm-ssd/ubuntu-jammy-22.04-amd64-server-20230919 The ec2 instance Amazon AMI name

Results

Go to your AWS EC2 Service console.
Your EC2 instance should be created with the name GL_NAMESPACE-GL_NAME-GL_STAGE-ec2

Try

You can use this KvBook to try to create an EC2 instance with KaiaC :
GL_KAIAC_MODULE="vmonly"
GL_NAMESPACE="vm001"
GL_NAME="demo1"
GL_STAGE="staging"
BE_PATH_EC2_PUBKEY="~/.ssh/id_rsa.pub"
BE_EC2_TYPE="t2.micro"
BE_EC2_VOLUME_SIZE="20"
BE_AMI_NAME="ubuntu/images/hvm-ssd/ubuntu-jammy-22.04-amd64-server-20230919"
