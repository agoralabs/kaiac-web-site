KvBook Amplifier

The Amplifier KvBook allow you to build Full-Stack Web and Mobile Apps Faster on AWS with KaiaC.

Design

This is what you can create with a CICD KvBook

Key/Value parameters

# Name Example value Description
1 GL_NAMESPACE cicd001 The application global resources namespace
2 GL_NAME demo1 The application name
3 GL_STAGE staging The application stage name
4 GL_KAIAC_MODULE amplifier The name of kaiac module to run
5 GL_REPO_TOKEN glpat-jCz5cxJKtmRfAj1a-NV2 Token to connect git repo
6 BE_FULL_REPOSITORY_ID https://gitlab.com/sky-scale-dev/kaiac/demo-kaiac-amplify.git Application git repo url
7 BE_SOURCE_BRANCH_NAME master The git branch name to deploy
8 GL_AWS_ROUTE53_DOMAIN skyscaledev.com AWS Amplify Domain Name
9 BE_FULL_REPOSITORY_ID agoralabs/demo-kaiac-laravel The application git repository id. This repository should contains at least a buildspec.yml file
10 FE_DNS demo-kaiac-amplify AWS Amplify Sub Domain Name
11 GL_AMPLIFY_PLATFORM WEB_COMPUTE Platform WEB(React and Angular) or WEB_COMPUTE (nextjs)
17 GL_INPUT_CONFIG_FILE /kaiac/001_amplifier_config.json Amplify JSON environment variables file : 
{{ '{' }}                                
    "_CUSTOM_IMAGE": "amplify:al2023",
    "NEXT_PUBLIC_API_ENDPOINT": "https://api-demo-kaiac.skyscaledev.com/api",
    "NEXT_PUBLIC_GOOGLE_MAP_API_ID": "customMapID",
    "NEXT_PUBLIC_GOOGLE_MAP_API_KEY": "SQGSSFGSFGfdsgjDummyKdgkkgdhdfkhdfkj"
{{ '}' }}

Results

Go to your AWS Amplify Service console.
Your Amplify application should be created

Try

You can use this KvBook to try to create an Amplify application with KaiaC :
GL_NAMESPACE="demo"
GL_NAME="kaiac"
GL_STAGE="staging"
GL_KAIAC_MODULE="amplifier"
GL_REPO_TOKEN="glpat-jCz5cxJKtmRfAj1a-NV2"
BE_FULL_REPOSITORY_ID="https://gitlab.com/sky-scale-dev/kaiac/demo-kaiac-amplify.git"
BE_SOURCE_BRANCH_NAME="master"
GL_AWS_ROUTE53_DOMAIN="skyscaledev.com"
FE_DNS="demo-kaiac-amplify"
GL_AMPLIFY_PLATFORM="WEB_COMPUTE"
GL_INPUT_CONFIG_FILE="/kaiac/001_amplifier_config.json"
