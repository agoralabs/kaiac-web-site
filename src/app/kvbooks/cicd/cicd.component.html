KvBook CICD

The CICD KvBook allow you to create an Application Deployment Pipeline with KaiaC.

Design

This is what you can create with a CICD KvBook

Key/Value parameters

# Name Example value Description
1 GL_AWS_REGION us-west-2 The AWS region your resources will be deployed
3 GL_AWS_CRED_FILE_PATH ~/.aws/credentials The AWS credentials file path
4 GL_AWS_CRED_PROFILE default The AWS credentials profile
5 GL_KAIAC_MODULE cicd The name of kaiac module to run
6 GL_NAMESPACE cicd001 The application global resources namespace
7 GL_NAME demo1 The application name
8 GL_STAGE staging The application stage name
9 GL_AWS_CODEBUILD_IMG aws/codebuild/standard:6.0 The AWS CodeBuild Image for compute build env
10 BE_EXISTING_EC2_DEPLOYMENT_GROUP vmgroup001-demo1-staging The Existing EC2 deployment group used by CodeDeploy
11 BE_ASG_EXISTING_NAME The Existing autoscaling group name is used
12 GL_CSTAR_CONN_NAME github_agora_conn The name of the codestar git repository connection
13 BE_FULL_REPOSITORY_ID agoralabs/demo-kaiac-laravel The application git repository id. This repository should contains at least a buildspec.yml file
14 BE_SOURCE_BRANCH_NAME main The git branch name to deploy
15 BE_PIPELINE_BUCKET cicd001-demo1-staging The name of S3 bucket to store source zip used in pipelines (will be created)
16 GL_DEPLOY_STAGE_ENABLED true True if deploy stage enabled - False if not
17 GL_INPUT_CONFIG_FILE /cicd_vm_config.json The environment variables config json file to inject in the pipeline :

Results

Go to your AWS CodePipeline Service console.
Your Pipeline should be created with the name GL_NAMESPACE-GL_NAME-GL_STAGE

Try

You can use this KvBook to try to create a CICD Pipeline with KaiaC :
GL_AWS_REGION="us-west-2"
GL_AWS_CRED_FILE_PATH="~/.aws/credentials"
GL_AWS_CRED_PROFILE="default"
GL_KAIAC_MODULE="cicd"
GL_SCRIPT_MODE="cloud"
GL_NAMESPACE="cicd001"
GL_NAME="demo1"
GL_STAGE="staging"
GL_AWS_CODEBUILD_IMG="aws/codebuild/standard:6.0"
BE_EXISTING_EC2_DEPLOYMENT_GROUP="vmgroup001-demo1-staging"
BE_ASG_EXISTING_NAME=""
GL_CSTAR_CONN_NAME="github_agora_conn"
BE_FULL_REPOSITORY_ID="agoralabs/demo-kaiac-laravel"
BE_SOURCE_BRANCH_NAME="main"
BE_PIPELINE_BUCKET="cicd001-demo1-staging"
GL_DEPLOY_STAGE_ENABLED="true"
GL_INPUT_CONFIG_FILE="/root/kaiac/kvbooks/cicd/001_runbook_cloud_oregon_cicd_vm_config.json"
