KvBook Swapper

The Swapper KvBook allow you to manage swap space to you ubuntu EC2 instance with KaiaC.

Design

This is what you can create with a Swapper KvBook

Key/Value parameters

# Name Example value Description
1 GL_AWS_REGION us-west-2 The AWS region your resources will be deployed
3 GL_AWS_CRED_FILE_PATH ~/.aws/credentials The AWS credentials file path
4 GL_AWS_CRED_PROFILE default The AWS credentials profile
5 GL_KAIAC_MODULE swapper The name of kaiac module to run
6 GL_NAMESPACE s3bucket001 The application global resources namespace
7 GL_NAME demo1 The application name
8 GL_STAGE staging The application stage name
9 BE_EXISTING_EC2_NAME vmonly001-demo1-staging-1 The Existing EC2 instance name to use
10 BE_EC2_SWAP_SIZE 250M The ec2 instance swap size to set

Results

Go to your EC2 Service console.
Connect through SSH to your instance and check the Swap using the command free -h

Try

You can use this KvBook to try to add swap with KaiaC :
GL_AWS_REGION="us-west-2"
GL_AWS_CRED_FILE_PATH="~/.aws/credentials"
GL_AWS_CRED_PROFILE="default"
GL_KAIAC_MODULE="swapper"
GL_NAMESPACE="swapper001"
GL_NAME="demo1"
GL_STAGE="staging"
BE_EXISTING_EC2_NAME="vmonly001-demo1-staging-1"
BE_EC2_SWAP_SIZE="250M"
